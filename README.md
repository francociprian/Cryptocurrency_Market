You need:

- [Node.js 16+ (recommend: 16.15.1 LTS)](https://nodejs.org/en/)

1. Clone the repository:


2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Run:

```bash
npm run dev
# or
yarn dev
```

4. Open [localhost:5173](localhost:5173) with your browser to see the result.
